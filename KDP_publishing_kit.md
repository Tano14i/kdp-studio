## 3. Checklist di assemblaggio finale

- [ ] **Manoscritto**: rileggere il DOCX, controllare refusi e impaginazione (i lettori
      puniscono duramente errori/traduzioni sciatte).
- [ ] **Front matter**: pagina titolo, copyright, eventuale dedica (già nel DOCX).
- [ ] **Trim size**: 6×9 pollici (15,24 × 22,86 cm).
- [ ] **Margini/bleed**: usare il template KDP per il numero di pagine effettivo.
- [ ] **Copertina**: generare immagine col prompt sopra → aggiungere testo → esportare
      PDF cover con dorso calcolato sul conteggio pagine.
- [ ] **eBook**: caricare DOCX o EPUB; verificare anteprima Kindle.
- [ ] **Categorie + keyword + prezzo**: inserire i valori della sezione 1.
- [ ] **⚠️ Dichiarazione AI**: nel workflow KDP, alla domanda sull'AI Content,
      dichiarare **Text: AI-generated** e (se la cover è AI) **Images: AI-generated**.
      L'editing sostanziale NON esonera dalla dichiarazione.
- [ ] **Anti-spam**: max 3 nuovi titoli al giorno per account.
- [ ] **Pubblica** (azione manuale dell'autore tramite login KDP).
